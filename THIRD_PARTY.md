# Third-party code and data

This repository vendors several third-party implementations and benchmark datasets so that the paper experiments can be replicated exactly. They remain under their own licenses, listed below. The repository's own code (workflow rules, wrappers, evaluation, the DOTS method) is covered by the top-level [LICENSE](LICENSE).

## Algorithm implementations

| Component | Location | Origin | License |
|---|---|---|---|
| DiffAN (adapted) | `workflow/scripts/algorithms/diffan/` | [vios-s/DiffAN](https://github.com/vios-s/DiffAN) (Sanchez et al., ICLR 2023) | See upstream repository |
| dodiscover (SCORE, CAM, DAS, NoGAM) | `workflow/scripts/algorithms/dodisc/dodiscover-main/` | [py-why/dodiscover](https://github.com/py-why/dodiscover) | MIT (`LICENSE` in that directory) |
| Tigramite (PCMCI, PCMCI+) | `workflow/scripts/algorithms/tigramite/tigramite-master/` | [jakobrunge/tigramite](https://github.com/jakobrunge/tigramite) | GPL-3.0 (`license.txt` in that directory) |
| TCDF | `workflow/scripts/algorithms/tcdf/TCDF-master/` | [M-Nauta/TCDF](https://github.com/M-Nauta/TCDF) (Nauta et al., 2019) | GPL-3.0 (`LICENSE` in that directory) |
| TiMINo | `workflow/scripts/algorithms/timino/codeTimino/` | [Jonas Peters](https://web.math.ku.dk/~peters/code.html) (Peters et al., NeurIPS 2013) | See `COPYING` in that directory |
| CAM pruning R code | `workflow/scripts/algorithms/diffan/pruning_R_files/` | Derived from the CAM R package (Bühlmann et al.) via the DiffAN repository | See upstream |

VARLiNGAM, DYNOTEARS, and the CI-test based baselines are used via their published packages (`lingam`, `causalnex`, `causal-learn`, `cdt`) declared in `workflow/envs/*.yml`, not vendored.

Note: because parts of the vendored code are GPL-3.0, redistribution of this repository as a whole may be subject to GPL-3.0 obligations.

## Datasets (`data/`)

| Dataset | Location | Origin |
|---|---|---|
| FinanceCPT | `data/FinanceCPT/` | Simulated financial time series by [Kleinberg](http://www.skleinberg.org/data.html), as used in the TCDF paper |
| NetSim fMRI | `data/fMRI/` | Simulated fMRI BOLD time series from [NetSim](https://www.fmrib.ox.ac.uk/datasets/netsim/) (Smith et al., NeuroImage 2011), as used in the TCDF paper |
| CausalTime (medical, pm25, traffic) | `data/causaltime/` | [CausalTime benchmark](https://www.causaltime.cc/) (Cheng et al., ICLR 2024) |
| ablation_lags | `data/ablation_lags/` | Generated by this repository (`config/paper_generate_data.yml`) |

Please cite the original sources when using these datasets.
